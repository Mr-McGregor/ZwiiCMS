Aide à la gestion des langues

1- Installer une langue disponible

L'installation d'une nouvelle langue provoque la création d'une nouveau jeu de pages.

Deux possibilités sont proposées :

  1. Commencez par choisir une langue. A ce stade, votre nouveau site ne comprendra que la page d'accueil. Vous aurez à créer de nouvelles pages et à les traduire.
  2. Pour vous éviter un fastidieux travail de mise en page, vous avez la possibilité de dupliquer un jeu de pages existant, il restera à traduire les textes dans la nouvelle langue.

2 - Installation d'une langue non prévue et de son drapeau :

  1. Soit la langue est disponible dans la liste prédéfinie, il suffit alors de supprimer les commentaires représentés par des barres obliques (//)
  2. Soit la langue n'est pas disponible, complétez le tableau en ajoutant

      3. Téléchargez sur le forum (rubrique téléchargement) une archive contenant une structure de données.

      Copiez et décompressez cette archive à la racine de votre stockage dans le gestionnaire de fichiers

      Un dossier i18n est créé, elle contient un fichier i18n.json, un dossier avec des drapeaux. Des langues supplémentaires sont proposées vous pouvez les modifier :

    1. Le code iso sur deux caractères en vous référant à cette page https://fr.wikipedia.org/wiki/Liste_des_codes_ISO_639-1
    2. Le nom de la langue en texte en clair, le rappel du code iso est facultatif. Le dernier élément de la liste ne doit pas se terminer par une virgule.
    3. Enfin, créez dans le dossier des drapeaux celui de la langue au format png avec les dimensions 30 pixels de large pour 20 pixels de haut. Le nom de l'image en minuscules correspond au code iso de la langue.
    4. Sans l'image du drapeau, la langue n'est pas affichée.
